INDUSTRY PROJECTS

Vendor Partner Gateway Severless Batch System

The Vendor Partner Gateway is a B2B gateway used to exchange information between Telstra and its B2B partners. Initially, VPG supports Telstra's Device Protect/Device upgrade. Technical point of view VPG is a collection of serverless applications (AWS lambda functions). VPG is used to update handset(mobile) related information on vendor servers using SFTP protocol. At present VPG application supports one vendor that buys old used phones and sells them in Asia. In this project, I have used AWS Lambdas and SQS queues cloudwatch, Splunk and SFTP server.

Bamboo CI/CD Migration to AWS (pipeline as a code)

I did CI/CD pipeline migration POC which helped 10 teams of order to activation department to migrate from existing Atlassian Bamboo server to Telstra's own Dockerized Bamboo servers. Pipeline as a code is created as a part of this project. I configured agents with necessary dependencies.

Order to Activation Microservices

Worked on microserviecs that power Telstra mobile services. Mobile suspend and Resume service: This service is responsible for suspending and resuming mobile services in case the cell phone is lost, stolen or payment overdue. Messagebank Services: These services provide a feature of leaving an audio message in case of a call is unanswered. The main categories are create, delete, modify and orchestrator service. In above-mentioned projects, I worked on development of springboot micorservices and used RabbitMq, Couchbase, APIGEE, Mokito, Bamboo CI/CD pipeline. The Services are deployed on AWS infrastructure and monitored through splunk, dynatrace and pagerduty.

ACADEMIC PROJECTS

Classical Planning for multiagent systems

During my M.Tech thesis under Dr. Indranil Saha, I have developed a planning tool which reduces multi-agent classical planning to SAT. It satisfies the obtained SAT problem and decodes the plan for the original problem. The tool name is WAVE and it guarantees to give the optimal make-span which make it first tool in the field of Multiagent Classical planning to give optimal makespan. The tool takes MAPDDL as an input and it uses its own custom parser to parse MAPDDL files. It beats state of the art Madagascar tool in terms of make-span..

Multiagent Sokoban planning

I worked on Multi-agent SOKOBAN SAT solver. The tool solves MA-Sokoban problem by generating the SAT clauses. The tool uses various clauses generation boosting techniques to solve the puzzle. The tool uses miniSAT tool as a black box for solution of SAT problems.

Type Ahead query suggestion System

When a user types a query in the search bar, all the relevant searched queries are recommended, if the search query changes dynamically (reduced or increased in length) so do the top K results.
